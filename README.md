# Generate-Powerset
The power set of S is the set of all possible subsets of S.

--input--

wq s

a b c

cbg y e tq

--output--

{} {s} {wq } {wq s}


{} {c} {b } {b c} {a } {a c} {a b } {a b c}


{} {tq} {e } {e tq} {y } {y tq} {y e } {y e tq} {cbg } {cbg tq} {cbg e } {cbg e tq} {cbg y } {cbg y tq} {cbg y e } 
{cbg y e tq}
